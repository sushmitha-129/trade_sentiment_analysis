# Trader Sentiment Analysis (Fear & Greed Index)

## Project Overview
This project analyzes the relationship between market sentiment (Fear & Greed Index) and trading performance using real trading data.

## Dataset
- Fear & Greed Index (daily sentiment scores)
- Historical trading data (PnL, trade size, volume)

## Objectives
- Understand how market sentiment impacts trading behavior
- Analyze profitability and trade volume under different sentiment conditions
- Derive actionable insights for traders

## Key Analysis
- Data cleaning and preprocessing
- Daily trade aggregation
- Sentiment classification (Fear / Neutral / Greed)
- Visual analysis using Seaborn & Matplotlib

## Key Insights
- Higher trading activity during Greed phases
- Increased volatility in PnL during extreme sentiment
- More stable returns during Neutral sentiment

## Actionable Recommendation
Traders should control leverage and trade size during Greed periods and explore conservative strategies during Fear phases.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
